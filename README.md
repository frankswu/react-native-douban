# react-native-douban

基于豆瓣开放平台实现的简易版豆瓣app。

### 技术栈

react: 15.2.1
react-native: 0.30.0
redux: 3.5.2
ES6

### 安装

```
    npm install
```

### 本地启动

```
    react-native run-ios
```
